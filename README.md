# üí∞ My Pet Project - Crypto Financial Tracker

**My Pet Project** is a Spring Boot application that allows users to track their cryptocurrency income and expenses. The main idea is to create one or more portfolios, to which transactions can be added. This allows tracking the financial history of each individual portfolio.

---

## üîß Technologies Used

- Java 17+
- Spring Boot
- Spring Data JPA
- PostgreSQL (or another relational database)
- REST API
- Maven
- dotenv
- Spring Security (JWT authentication)

---

## üß© Project Structure

- `config/` ‚Äî security configuration (`SecurityConfig.java`)
- `controller/` ‚Äî REST controllers for users and portfolios
- `dao/` ‚Äî DAO interfaces for working with the database
- `dotenv/` ‚Äî configuration for working with `.env` files
- `dto/` ‚Äî DTO classes for transferring data between Entity classes and the outside world
- `entity/` ‚Äî Entity classes
- `exception_handling/` ‚Äî Global API exception handler
- `filter/` ‚Äî JWT filter (`JwtFilter.java`) for processing tokens
- `handler/` ‚Äî custom security handlers
- `service/` ‚Äî Business logic for users and portfolios
